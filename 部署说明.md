# 仓库触摸看板 — 部署说明

## 📁 文件夹结构

```
warehouse-map/
├── index.html      ← 主页面（直接双击用浏览器打开）
├── data.json       ← 数据文件
└── 数据库.xlsx     ← Excel 模板（可选）
```

---

## 🚀 当前功能演示

已实现 **A区货架** 的完整流程：

```
点击货架 A
    ↓
展开层列表（第1/2/3/4层）
    ↓
点击层 → 显示货位网格
    ↓
点击货位 → 弹窗显示物料详情
```

---

## 🔧 如何添加更多货架

在 `index.html` 中，找到 `<!-- 货架热区示例 -->` 部分，复制以下代码：

```html
<div class="shelf" id="shelf-b" onclick="showLayers('B')">
  <div class="shelf-title">◈ B 区货架</div>
  <div class="shelf-sub">4层 · 每层6-20个货位</div>
  <div class="shelf-layers">点击进入</div>
  <div class="layer-row" id="shelf-b-preview"></div>
</div>
```

同时在 `renderPreviews()` 函数中加一行：
```javascript
['A', 'B'].forEach(shelf => { ... });
```

## 📋 数据格式说明

每条数据对应一个**货位**：

| 字段 | 说明 | 示例 |
|------|------|------|
| 库位编码 | 货架-层-位，固定格式 | A-L1-P001 |
| 物料名称 | 物料名称，留空=空库位 | 螺丝 M8×30 |
| 规格型号 | 物料规格 | M8-30-SS |
| 数量 | 数字，0=空库位 | 120 |
| 单位 | 个/盒/件/箱/kg | 盒 |
| 物料图片路径 | 图片路径，留空无图片 | /images/photo.jpg |
| 备注 | 任意备注文字 | 常用规格 |

**编码规则：** `货架-层-位`
- 货架：A/B/C/D...
- 层：L1/L2/L3/L4
- 货位：P001 ~ P999（最多支持999个）

## 🔄 数据更新流程

1. 编辑 `data.json` 或使用 Excel 模板
2. 保存后按 **F5** 刷新页面即可

---

## 🎯 触屏部署

1. 复制 `warehouse-map` 文件夹到触屏电脑
2. 用 Edge 打开 `index.html`
3. **F11** 全屏显示
4. 任务栏设为自动隐藏
5. 完成 ✅

## ⚙️ 状态颜色说明

| 颜色 | 含义 | 条件 |
|------|------|------|
| 🟢 绿 | 正常 | 数量 > 50 |
| 🟡 黄 | 库存不足 | 10 < 数量 ≤ 50 |
| 🔴 红（闪烁）| 严重告警 | 数量 ≤ 10 |
| ⬜ 灰 | 空库位 | 无物料或数量=0 |
